## 📊 ملخص المشروع

**حراج اليمن** هو مشروع تقني متكامل يهدف إلى إنشاء منصة إعلانات مبوبة فاخرة متخصصة في السوق اليمني، بنيت بتقنيات الويب الحديثة وهندسة معمارية مفصولة (Decoupled Architecture) تضمن الأداء العالي والقابلية للتوسع.

### الإحصائيات الرئيسية:
- **إجمالي الملفات:** 37 ملف (PHP، JS، CSS، MD)
- **إجمالي الأسطر البرمجية:** ≈ 3,000+ سطر
- **جداول قاعدة البيانات:** 12 جدول متكامل
- **واجهات API:** 8 واجهات برمجية رئيسية
- **صفحات المستخدم:** 10 صفحات تفاعلية

### 2.2 تحليل الجداول الرئيسية

#### 2.2.1 جدول المستخدمين (users)
```sql
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    phone VARCHAR(20) UNIQUE NOT NULL,        -- مفتاح فريد للهاتف
    email VARCHAR(100) DEFAULT NULL,
    password VARCHAR(255) NOT NULL,
    avatar VARCHAR(255) DEFAULT NULL,
    role VARCHAR(20) DEFAULT 'user',         -- admin/user
    rating FLOAT DEFAULT 5.0,                -- نظام التقييم 1-5
    isBanned TINYINT(1) DEFAULT 0,          -- حالة الحظر
    joinedDate VARCHAR(20) NOT NULL,         -- تاريخ الانضمام (YYYY-MM)
    createdAt DATETIME DEFAULT CURRENT_TIMESTAMP
) ENGINE=InnoDB;
```

**الملاحظات:**
- استخدام `phone` كمفتاح فريد بدلاً من البريد الإلكتروني (مناسب للسوق اليمني)
- نظام التقييم مضمن في الجدول الرئيسي
- دعم متعدد الأدوار (admin/user)

#### 2.2.2 جدول الإعلانات (ads)
```sql
CREATE TABLE ads (
    id INT AUTO_INCREMENT PRIMARY KEY,
    userId INT NOT NULL,
    title VARCHAR(255) NOT NULL,
    description TEXT NOT NULL,
    category VARCHAR(50) NOT NULL,
    city VARCHAR(50) NOT NULL,
    price DECIMAL(12,2) DEFAULT NULL,
    images JSON,                            -- تخزين الصور كـ JSON
    specifications JSON,                     -- المواصفات التفصيلية
    carBrand VARCHAR(50),                    -- ماركة السيارة
    carYear VARCHAR(4),                      -- سنة الصنع
    carTransmission VARCHAR(20),             -- ناقل الحركة
    carMileage INT,                          -- المسافة المقطوعة
    propertyType VARCHAR(50),                 -- نوع العقار
    propertyRooms VARCHAR(20),               -- عدد الغرف
    propertyContract VARCHAR(50),            -- نوع العقد
    views INT DEFAULT 0,                     -- عدد المشاهدات
    isPinned TINYINT(1) DEFAULT 0,          -- تثبيت الإعلان
    status ENUM('active', 'sold', 'deleted') DEFAULT 'active',
    createdAt DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (userId) REFERENCES users(id) ON DELETE CASCADE
) ENGINE=InnoDB;
```

**الميزات المتقدمة:**
- تخزين الصور كـ JSON للتعامل مع متعدد الصور
- تخزين المواصفات كـ JSON لدعم الخصائص الديناميكية
- نظام تثبيت الإعلانات (pinned ads)
- تتبع المشاهدات (views counter)

function jsonSuccess($data = [], $message = 'تمت العملية بنجاح') {
    echo json_encode([
        'success' => true,
        'message' => $message,
        'data'    => $data
    ], JSON_UNESCAPED_UNICODE | JSON_PRETTY_PRINT);
    exit;
}

function jsonError($message = 'حدث خطأ', $code = 400) {
    http_response_code($code);
    echo json_encode([
        'success' => false,
        'message' => $message,
        'data'    => null
    ], JSON_UNESCAPED_UNICODE | JSON_PRETTY_PRINT);
    exit;
}

### 7.2 فحص صحة النظام
```php
// check_db.php - فحص جميع الجداول والعلاقات
try {
    $tables = ['users', 'ads', 'comments', 'favorites', 'messages', 
               'chat_threads', 'reviews', 'notifications', 'reports', 
               'commission_transfers'];
    
    foreach ($tables as $table) {
        $count = $db->query("SELECT COUNT(*) FROM $table")->fetchColumn();
        echo "✓ $table: $count سجل\n";
    }
    
    // فحص العلاقات الأجنبية
    $foreignKeys = $db->query("
        SELECT TABLE_NAME, COLUMN_NAME, CONSTRAINT_NAME, 
               REFERENCED_TABLE_NAME, REFERENCED_COLUMN_NAME
        FROM INFORMATION_SCHEMA.KEY_COLUMN_USAGE
        WHERE TABLE_SCHEMA = 'haraj_db' 
        AND REFERENCED_TABLE_NAME IS NOT NULL
    ")->fetchAll();
    
    echo "\n🔗 العلاقات الأجنبية:\n";
    foreach ($foreignKeys as $fk) {
        echo "  {$fk['TABLE_NAME']}.{$fk['COLUMN_NAME']} → {$fk['REFERENCED_TABLE_NAME']}.{$fk['REFERENCED_COLUMN_NAME']}\n";
    }
} catch (Exception $e) {
    echo "❌ خطأ: " . $e->getMessage() . "\n";
}
```
