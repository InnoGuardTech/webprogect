# 🇾🇪 حراج اليمن الفاخر - الإصدار المتقدم

هذا هو المستودع الرسمي لمنصة "حراج اليمن"، أكبر وأحدث منصة مبوبة للإعلانات في اليمن، مبنية بتقنيات الويب الحديثة بهندسة معمارية مفصولة (Decoupled Architecture).

## 🌟 الميزات التقنية (Tech Stack)
- **البنية التحتية:** PHP 8+ و MySQL.
- **الواجهات الأمامية:** HTML5, CSS3 المخصص (Glassmorphism & Micro-animations)، و Vanilla JavaScript (Fetch API).
- **الواجهات الخلفية:** JSON API مخصص ومحمي بنظام الجلسات (Sessions)، جاهز للتوسع ليدعم تطبيقات الجوال (Mobile Apps) مستقبلاً.
- **التصميم:** متجاوب (Responsive) بالكامل، يدعم الوضع الداكن (Dark Mode) افتراضياً.

## 📂 هيكل النظام

```
/
├── backend/            # طبقة الـ API والخوادم
│   ├── config.php      # إعدادات قاعدة البيانات والأمان
│   ├── router.php      # موجه مسارات API (Dispatcher)
│   ├── auth.php        # واجهات تسجيل الدخول والإنشاء
│   ├── ads.php         # جلب ونشر الإعلانات (مرفقات Base64)
│   ├── cities.php      # المدن اليمنية (صنعاء، عدن...)
│   ├── categories.php  # الأقسام والتصنيفات
│   └── admin.php       # عمليات الإدارة المتقدمة
│
├── frontend/           # طبقة عرض البيانات للمستخدم
│   ├── assets/
│   │   ├── css/style.css # نظام التصميم المتكامل (Premium Design)
│   │   └── js/app.js     # الوظائف الحيوية (AJAX, Toasts, Dark Mode)
│   ├── index.php       # الصفحة الرئيسية
│   ├── auth.php        # بوابة الدخول الموحدة
│   ├── post.php        # نظام نشر الإعلانات الديناميكي
│   ├── ad.php          # صفحة تفاصيل الإعلان
│   ├── admin.php       # لوحة تحكم SPA سريعة
│   └── commission.php  # بوابة العمولات والبنوك المحلية
│
└── scripts/            # أدوات الصيانة للمطور
    └── db_setup.php    # تثبيت وبناء قاعدة البيانات من الصفر
```

## 🚀 كيفية التشغيل محلياً
1. ضع المشروع في مجلد `www` في الـ `WampServer`.
2. تأكد من عمل سيرفر `Apache` و `MySQL`.
3. قم بزيارة: `http://localhost/aaaa/scripts/db_setup.php` لتأسيس قاعدة البيانات.
4. اذهب إلى `http://localhost/aaaa/frontend/index.php` لتصفح الموقع.

## 🛡️ الأمان والحماية
- يتم استخدام `PDO` لمنع الحقن (SQL Injection).
- يتم تمرير جميع المدخلات عبر دالة مخصصة لفلترة البيانات لمنع (XSS).
- تشفير كلمات المرور باستخدام أحدث دوال `password_hash`.
- نظام Base64 لرفع الصور يحمي السيرفر من الملفات الخبيثة (Web Shells).
